# WolfberryWords — Word of the Day Widget

> *A glossary in progress — logging the words that made me stop and reach for a pencil.*

A free iPhone home screen widget that displays a beautiful, unusual literary term every single day. Built for word lovers, readers, and anyone who has ever dog-eared a page just for a sentence.

---

## What It Looks Like

A dark academia-styled widget that shows:

- **The word** in an elegant serif font
- **Phonetic pronunciation** so you know how to say it
- **Part of speech** — noun, verb, adjective, or adverb
- **Definition** — concise and beautifully worded

A new word appears automatically every morning. No tapping, no opening an app. Just words.

---

## How to Install

**You will need the free Scriptable app. No account required. No data collected.**

### Step 1 — Download Scriptable
Get the free [Scriptable app](https://apps.apple.com/us/app/scriptable/id1405459188) from the App Store.

### Step 2 — Add the Script
1. Open Scriptable
2. Tap **+** in the top right corner (or **Edit → Add Widget** on iPhone 16 / iOS 18)
3. Copy the entire contents of [WolfberryWords.js](./WolfberryWords.js) — tap the file, then tap **Raw**, then select all and copy
4. Paste into Scriptable
5. Name it **WolfberryWords**
6. Tap **Done**

### Step 3 — Add to Your Home Screen
1. Long-press your iPhone home screen until icons wiggle
2. Tap **Edit** → **Add Widget** (iOS 18) or tap **+** (iOS 17 and below)
3. Search for **Scriptable**
4. Select **Medium** size
5. Tap **Add Widget**
6. Tap the widget and select **WolfberryWords** as the script
7. Tap **Done**

That's it. A new word appears every morning automatically.

## Changing the Theme

Open the script in Scriptable and find this line near the top:

```javascript
const THEME = "original";
```

Change `"original"` to any of these options:

| Theme | Description |
|---|---|
| `original` | Warm dark brown (default) |
| `forest` | Deep forest green |
| `blood` | Oxford blood red |
| `charcoal` | Deep charcoal |
| `evergreen` | Dark teal green |
| `black` | Pure black |
| `ivory` | Light parchment (light mode) |
